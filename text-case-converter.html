Case Conversion Examples

UPPERCASE

Converts all letters to uppercase. Useful for headings, titles, or emphasis.

Original: hello world → UPPERCASE: HELLO WORLD

lowercase

Converts all letters to lowercase. Useful for normalizing text or code.

Original: Hello World → lowercase: hello world

Title Case

Capitalizes the first letter of each word. Useful for titles, headings, and proper nouns.

Original: hello world → Title Case: Hello World

Sentence case

Capitalizes the first letter of each sentence. Useful for normal paragraphs.

Original: hello. world! → Sentence case: Hello. World!
