Compression Tips

  • Low Compression: Best for documents with important images or graphics
  • Medium Compression: Ideal for standard documents with some images
  • High Compression: Perfect for text-heavy documents or web use
  • Use advanced settings for more control over image optimization

About PDF Compression

  • Compression reduces file size by optimizing images and removing unnecessary data
  • Text and vector graphics remain crisp at any compression level
  • Compressed PDFs maintain full functionality (text search, links, etc.)
  • For actual PDF compression in production, a server-side API would be required
