About UUIDs

What is a UUID?

A UUID (Universally Unique Identifier) is a 128-bit number used to uniquely identify information in computer systems. The probability of generating duplicate UUIDs is vanishingly small.

UUID Versions

Version 4 (Random): Generated using random or pseudo-random numbers. Most common type.

Version 1 (Timestamp): Generated using timestamp and MAC address of the computer.

Common Uses

• Database primary keys

• Session identifiers

• Tracking unique transactions

• File naming and versioning

• Distributed systems
